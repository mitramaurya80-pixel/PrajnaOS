01
What is Video Memory?
›

Video memory (also called VRAM) is a special region of RAM that the VGA chip watches constantly. Whatever you write into this memory region appears on the screen automatically — no system calls, no OS needed.

In text mode, video memory starts at address 0xB8000. In graphics mode (mode 13h), it starts at 0xA0000.

0x00000640 KBconventional RAM (your kernel lives here)
0xA000064 KBVGA graphics memory (mode 13h pixels)
0xB80004 KBVGA text memory ← WE USE THIS
0xC0000256 KBBIOS ROM and hardware reserved
Key rule: You write to 0xB8000 in C using a pointer cast:
char *vga = (char *)0xB8000;
The VGA chip reads this memory 60 times per second and draws it on screen!
02
The 2-Byte Cell Format
›

The screen in text mode is a 80 columns × 25 rows grid = 2000 cells total. Each cell takes exactly 2 bytes in memory:

/* Cell layout — 2 bytes per character on screen */ /* */ /* Byte 0: the ASCII character (e.g. 'H' = 0x48) */ /* Byte 1: the color byte (e.g. 0x07 = white) */ char *vga = (char *)0xB8000; vga[0] = 'H'; /* character at position 0 */ vga[1] = 0x07; /* color: white on black */ vga[2] = 'i'; /* character at position 1 */ vga[3] = 0x07; /* color for position 1 */

Using uint16_t lets you write both bytes in one operation:

uint16_t *vga = (uint16_t *)0xB8000; /* high byte = color, low byte = character */ vga[0] = (0x07 << 8) | 'H'; /* both in one write! */
Total size: 80 × 25 × 2 = 4000 bytes. Your entire screen fits in 4KB of memory at 0xB8000!
03
The Color Byte — 16 Colors
›

The color byte is split into two 4-bit halves. High 4 bits = background color, low 4 bits = foreground (text) color.

/* color byte = 0xBF where B=background F=foreground */ /* 0x07 = 0000 0111 = black bg + light gray text */ /* 0x0A = 0000 1010 = black bg + bright green text */ /* 0x1F = 0001 1111 = blue bg + white text */ uint8_t make_color(uint8_t fg, uint8_t bg) { return (bg << 4) | fg; /* shift bg to high 4 bits, put fg in low 4 bits */ }

04
Row and Column — Calculating Position
›

The screen is a flat array in memory, but we think of it as a 2D grid. To write to row R, column C:

/* formula: position = row * 80 + col */ /* because each row has exactly 80 characters */ void put_char_at(char c, uint8_t color, int row, int col) { uint16_t *vga = (uint16_t *)0xB8000; vga[row * 80 + col] = (color << 8) | (uint8_t)c; } /* examples: */ put_char_at('A', 0x0A, 0, 0); /* top-left corner */ put_char_at('Z', 0x0F, 24, 79); /* bottom-right */ put_char_at('X', 0x04, 12, 40); /* middle of screen */
Why row * 80? Each row has 80 columns. To skip 3 rows: 3 × 80 = 240 cells. Then add the column number to reach the exact cell.
05
Printing Strings and Cursor Tracking
›

To print full strings we need to track where the cursor is and advance it after each character. We also need to handle special characters like newline.

static int cursor_row = 0; /* current row (0-24) */ static int cursor_col = 0; /* current col (0-79) */ void putchar(char c) { if (c == '\n') { /* newline → next row */ cursor_row++; cursor_col = 0; } else if (c == '\b') { /* backspace → go back */ if (cursor_col > 0) cursor_col--; put_char_at(' ', 0x07, cursor_row, cursor_col); } else { put_char_at(c, 0x07, cursor_row, cursor_col++); if (cursor_col >= 80) { /* wrap to next line */ cursor_col = 0; cursor_row++; } } }
06
Clear Screen and Scrolling
›

When the screen fills up we need to scroll — move every row up by one and clear the last row. Clearing the screen fills all cells with spaces.

void clear_screen() { uint16_t *vga = (uint16_t *)0xB8000; for (int i = 0; i < 80*25; i++) vga[i] = (0x07 << 8) | ' '; /* fill all 2000 cells with space + white color */ cursor_row = cursor_col = 0; } void scroll() { uint16_t *vga = (uint16_t *)0xB8000; /* copy each row to the row above it */ for (int i = 0; i < 80*24; i++) vga[i] = vga[i + 80]; /* clear the last row */ for (int i = 80*24; i < 80*25; i++) vga[i] = (0x07 << 8) | ' '; cursor_row = 24; /* cursor goes to last row */ }
07
Hardware Cursor — Real Blinking Cursor
›

The VGA chip has a real hardware cursor that blinks automatically. You control its position by writing to VGA I/O ports using the outb instruction.

/* VGA ports: */ /* 0x3D4 = index register (tell VGA what to write) */ /* 0x3D5 = data register (write the value) */ void outb(uint16_t port, uint8_t val) { __asm__ volatile ("outb %0, %1" :: "a"(val), "Nd"(port)); } void move_cursor(int row, int col) { uint16_t pos = row * 80 + col; outb(0x3D4, 14); /* select high byte reg */ outb(0x3D5, pos >> 8); /* send high byte */ outb(0x3D4, 15); /* select low byte reg */ outb(0x3D5, pos & 0xFF); /* send low byte */ }
08
Printing Numbers Without printf
›

In your OS you have no printf, no stdlib. You must write your own number-to-string conversion. Here's how to print hex and decimal numbers:

/* print a 32-bit number as hex: e.g. 0xDEADBEEF */ void print_hex(uint32_t n) { char digits[] = "0123456789ABCDEF"; putchar('0'); putchar('x'); for (int i = 28; i >= 0; i -= 4) { /* shift right by i bits, mask lower 4 bits */ putchar(digits[(n >> i) & 0xF]); } } /* print decimal: convert digit by digit */ void print_dec(uint32_t n) { char buf[10]; int i = 0; if (n == 0) { putchar('0'); return; } while (n > 0) { buf[i++] = '0' + (n % 10); n /= 10; } while (i-- > 0) putchar(buf[i]); /* print reversed */ }
09
VGA Graphics Mode — Drawing Pixels
›

Mode 13h switches VGA to 320×200 pixels with 256 colors. Each pixel = 1 byte. Memory starts at 0xA0000 instead of 0xB8000.

/* pixel memory in mode 13h */ uint8_t *pixels = (uint8_t *)0xA0000; /* draw one pixel at x,y with color (0-255) */ void put_pixel(int x, int y, uint8_t color) { pixels[y * 320 + x] = color; /* y*320 = skip y rows (320 pixels per row) */ } /* draw a horizontal line */ void hline(int x, int y, int len, uint8_t c) { for (int i = x; i < x+len; i++) put_pixel(i, y, c); } /* fill screen with one color */ void fill_screen(uint8_t c) { for (int i = 0; i < 320*200; i++) pixels[i] = c; }

Common Mistakes to Avoid
›
/* MISTAKE 1: forgetting to multiply by 2 with char* */ char *vga = (char *)0xB8000; vga[1] = 'B'; /* WRONG! index 1 = the COLOR of char 0 */ vga[2] = 'B'; /* correct — index 2 = char at position 1 */ /* MISTAKE 2: using int instead of uint16_t */ int *vga = (int *)0xB8000; /* WRONG! int=4 bytes, skips 2 chars */ uint16_t *vga = (uint16_t *)0xB8000; /* correct */ /* MISTAKE 3: wrong row formula */ vga[row + col] = ...; /* WRONG! misses row width */ vga[row * 80 + col] = ...; /* correct */ /* MISTAKE 4: color byte order wrong */ vga[i] = (fg << 8) | bg; /* WRONG! fg and bg swapped */ vga[i] = (bg << 8) | fg; /* correct: bg in high bits */ /* MISTAKE 5: not checking screen boundaries */ /* if row >= 25 → writing past screen = random crash */ if (row >= 25) scroll(); /* always check! */
